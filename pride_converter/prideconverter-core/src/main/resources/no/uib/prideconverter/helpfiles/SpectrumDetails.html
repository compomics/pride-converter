Each spectrum can be annotated by CV terms and user parameters.

To add a CV term click on the "Ontology Lookup Service" button. This will open a dialog with a graphical interface to the ontology lookup service.

For some of the CV terms you may want to add additional information. This is done by clicking in the "Value" column and inserting the given value. Please note that after inserting the value you have to click Enter to complete the insertion.

Adding user parameters is done by clicking the "Add User Parameter" button below the User Parameters table.

Right clicking in on of the tables opens a popup menu that allows you to edit or delete the rows in the table, and change the order of the rows. Editing a row can also be done by double clicking on the given row.
