ProjectSelection

In ms_lims the data is stored as projects. You can only convert data from one ms_lims project at the time. If you don't know the name of your project trying clicking on the most likely project titles and read the project descriptions appearing at the lower half of the screen. If you still can not find the right project contact the people responsible for your ms_lims installation and see if they can help you locate your project.

The project table can be sorted according to the contents of the different columns by clicking on the column titles.
