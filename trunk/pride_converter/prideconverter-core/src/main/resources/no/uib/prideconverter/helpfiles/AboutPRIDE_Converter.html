About PRIDE Converter

PRIDE Converter has been developed by Harald Barsnes (PhD student at the University of Bergen, Norway) under the guidance of Dr. Lennart Martens (EBI).

As of July 2009 further development and maintenance of PRIDE Converter has been turned over to the the PRIDE team at the EBI. To contact the team send an e-mail to pride-support@ebi.ac.uk.

Project home page: http://pride-converter.googlecode.com

PRIDE Converter is based on three other projects:
  • ms_lims
  • PRIDE
  • OLS

In addition:
  • Mascot dat file parsing is done using the MascotDatfile Library
  • iTRAQ calculation is based on a translation of the PERL code from i-Tracker
  • The platform independent graphical user interface uses code from JGoodies
  • Advanced JTables from SwingLabs SwingX
  • mzXML parsing is done using JRAP
  • mzML parsing is done using jmzml
  • OMSSA parsing is done using the omssa-parser

Help:

Click the help icons in the bottom left corner of each frame for specific help, or see the Troubleshooting section at http://pride-converter.googlecode.com for more general tips and hints.

Any further questions can be forwarded to the PRIDE team at the EBI: pride-support@ebi.ac.uk.

The converter has been tested on MS Windows XP, MS Windows Vista, Linux and Mac. And should also work on older Windows versions and on other platforms, as long as Java 1.5 (or newer) is installed.
